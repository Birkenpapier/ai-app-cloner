# Code of Conduct

Be decent. This is a small open-source project run by humans on their own time.

- **Assume good faith.** Half the "bugs" in someone's clone are just a screenshot
  the tool measured wrong. Start from there.
- **Critique the code, not the person.** No harassment, discrimination, or personal
  attacks.
- **Keep it legal.** Do not use this project, or its issues and PRs, to coordinate
  ripping off someone else's product. See [Legal and ethics](README.md#legal-and-ethics).

If someone is making the space worse, email the maintainer
([Birkenpapier](https://github.com/Birkenpapier)) and it will be handled.
